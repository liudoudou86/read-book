# 专属 AI Skill 生成模板

## 目标

将书籍的核心知识封装为一个可调用的 AI Skill，后续可以基于这本书进行问答、复习、深入探讨。

## 输入

基于 Step 5 生成的读书笔记，提取核心信息。

## 生成步骤

### 第一步：提取核心知识

从读书笔记中提取：

1. **核心观点**（3-5条）
2. **关键概念**（5-10个）
3. **适用场景**
4. **边界/局限**
5. **与其他知识的关联**

### 第二步：设计 Skill 能力

这个 Skill 应该能够：
- 回答关于这本书的问题
- 解释书中的核心概念
- 根据书中的方法提供建议
- 推荐书中方法的适用场景

### 第三步：生成 Skill 文件

## Skill 文件模板

```markdown
---
name: book-{slug}
description: "{书名} - 读书笔记助手，基于《{书名}》的专属 AI 助手"
user-invocable: true
---

## 使用说明

当你需要以下帮助时调用本 Skill：
- 想了解《{书名}》的核心内容
- 遇到{主题}相关的问题，想参考书中的方法
- 想深入理解{概念}的含义
- 想把这本书的知识应用到实际场景
```

## 输出路径

- 目录：`books/{slug}/`
- 文件：`SKILL.md`

## 命名规则

- slug = 书名全拼 + 出版年（仅保留英文/数字/短横）
- 示例：《如何阅读一本书》→ `book-how-to-read-a-book`
- 示例：《刻意练习》→ `book-deliberate-practice-2016`

## 注册 Skill

生成后，需要将 Skill 注册到系统中，使其可通过触发词调用。

**触发词**：
- `/book-{slug}` - 完整版
- `/book-{short-slug}` - 短版（取书名前3-4个词）
