# 专属 AI Skill 生成模板 — RIA++ 版本

## 目标

将书籍的核心方法论封装为一组可被 Agent 精准调用的 AI Skill。每个方法论单元以 RIA++ 六段结构呈现。

## 输入

基于 Step 5 生成的读书笔记（含方法论拆解），通过脚本自动生成。

## 生成方式

```bash
# 自动生成（含三重验证过滤）
uv run python scripts/skill_generator.py \
  --book-title "{书名}" \
  --note-path "~/.opencode/skill/books/{slug}/note.md"
```

脚本执行：
1. 读取笔记中的方法论单元列表
2. 执行三重验证（V1跨域 / V2预测力 / V3独特性）
3. 三 ✓ 的单元 → SKILL.md 的 RIA++ 卡片
4. 含 ✗ 的单元 → `rejected/{单元名}.md`（附原因）
5. 按 RIA++ 模板生成结构化 SKILL.md

## SKILL.md 输出结构

```markdown

name: book-{slug}
description: "{书名} — 基于 RIA++ 方法论的读书助手。适用于：{A2 触发场景摘要}"
user-invocable: true
use_case: |
  当用户询问关于《{书名}》的方法论、决策框架、可执行步骤时调用。
  语言信号：{A2 语言信号摘要}
---
```

## 输出路径

- 目录：`books/{slug}/`
- 文件：`SKILL.md`

## 命名规则

- slug = 书名全拼 + 出版年（仅保留英文/数字/短横）
- 示例：《如何阅读一本书》→ `book-how-to-read-a-book`
- 示例：《刻意练习》→ `book-deliberate-practice-2016`

## 注册 Skill

生成后，需要将 Skill 注册到系统中，使其可通过触发词调用。

**触发词**：
- `/book-{slug}` - 完整版
- `/book-{short-slug}` - 短版（取书名前3-4个词）
