# 关键词提取指导

## 目标

与作者「达成共识」——理解作者使用的关键概念，确保你们对同一词汇有相同的理解。

## 为什么要找关键词？

作者与你之间隔着一道「文字密码」。只有破解这些密码（关键概念），你才能真正理解作者的思想。

## 找什么？

### 1. 反复出现的词

如果在书中某个词反复出现多次，这个词通常是关键概念。

### 2. 专业术语

作者明确定义的术语，如：
- 经济学中的「供给」「需求」
- 心理学中的「认知」「行为」
- 本书中的「检视阅读」「分析阅读」

### 3. 作者赋予特殊含义的词

作者可能用日常词汇但赋予特殊含义。

## 如何确认理解一致？

### 检验方法一：用自己的话复述

> 找出关键的句子，然后用自己的话说一遍。如果说不出，说明你没有真正理解。

### 检验方法二：举一个例子

> 能否举一个例子来说明这个概念？如果举不出，说明理解不透彻。

### 检验方法三：查找定义

> 看看作者在哪里对这个词做了定义或解释。

## 提取步骤

### 第一步：列出关键词

找出 5-10 个你认为重要的关键词。

### 第二步：确认定义

对每个关键词，回答：
- 作者是如何定义的？
- 我认同这个定义吗？

### 第三步：找出核心主旨

> 书的作者总是借助一些句子来表达他的观点。这些重要的句子就是「主旨句」。

找出作者最核心的 3-5 个主张（主旨句）。

## 输出格式

```markdown
## 核心概念

| 概念 | 作者的定义 | 我的理解 | 认同度 |
|------|-----------|---------|--------|
| 概念1 | ... | ... | ✓ / △ / ✗ |
| 概念2 | ... | ... | ✓ / △ / ✗ |

## 核心主旨

1. **主旨1**
   - 原文：[作者的原话]
   - 我的理解：[用自己的话]
   
2. **主旨2**
   - ...

## 关键词出现的章节
- 概念1：主要在第X章讨论
- 概念2：主要在第Y章讨论
```

## 注意事项

- 不要过度细分，理解主要概念即可
- 重点关注：作者在书中反复强调的概念
- 不是所有术语都重要，聚焦核心
