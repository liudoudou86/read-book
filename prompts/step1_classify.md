# 书籍分类指导

## 为什么需要先分类？

不同类型的书籍需要不同的分析方法。读小说和读哲学书的方法完全不同。在开始深度阅读之前，先判断书籍类型，可以节省大量时间。

## 分类方法

### 1. 虚构 vs 非虚构

**虚构小说类**：
- 故事、戏剧、诗歌
- 阅读重点：情节、人物、情感体验
- 分析方法：沉浸式阅读，关注故事线和人物成长

**非虚构论述类**：其他所有书

### 2. 非虚构类的细分

#### 理论性作品
- **历史书**：讲述过去发生的事
  - 阅读重点：时间线、因果关系、历史意义
- **科学书**：关于自然世界的知识
  - 阅读重点：实验、证据、结论的可靠性
- **哲学书**：关于存在、认识、价值的问题
  - 阅读重点：论证、概念、逻辑推理
- **数学书**：关于数量和空间的知识
  - 阅读重点：公理、定理、推导过程

#### 实用性作品
- 告诉你「应该做什么」或「如何做」
- 实用书类型：
  - 自我提升类
  - 方法论类
  - 商业/经济类
- **关键检验**：作者是否提出了可检验的方法？

#### 社会科学类
- 人类学、经济学、社会学、政治学等
- 特点：涉及多个学科的概念
- 阅读重点：关注作者的观点和立场

### 3. 快速判断技巧

1. **看副标题**：实用书通常有「...的指南」「...方法」「如何...」
2. **看目录**：理论书目录往往呈现「问题-论证」结构
3. **看章节标题**：如果章节标题是问句，可能是哲学/科学类

## 输出格式

完成分类后，记录：

```markdown
## 书籍分类

### 类型判断
- 大类：[虚构小说 / 理论性作品 / 实用性作品 / 社会科学]
- 细分：[历史 / 科学 / 哲学 / 数学 / 自我提升 / ...]

### 分析方法选择
- 重点关注：[如：论证逻辑 / 可执行方法 / 故事结构]
- 阅读速度建议：[快读 / 细读 / 反复研读]
```
