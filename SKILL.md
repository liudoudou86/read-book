> 本 Skill 基于《如何阅读一本书》的分析阅读法，帮助你深度分析一本书籍，输出结构化读书笔记，并生成专属 AI Skill。

## 触发条件

当用户说以下内容时启动：
- `/read-book`
- "帮我读一本书"
- "分析这本书"
- "生成读书笔记"
- 上传 `.txt` 或 `.pdf` 文件

当用户想继续基于已分析的书籍进行问答时：
- 调用生成的专属 Skill（如 `/book-xxx`）

---

## 工具使用规则

| 任务            | 使用工具                                                      |
| --------------- | ------------------------------------------------------------- |
| 读取书籍内容    | `Read` 工具（`.txt`）或 `Bash` + `pdf_extractor.py`（`.pdf`） |
| 写入读书笔记    | `Write` 工具                                                  |
| 追加评价/思考   | `Edit` 工具                                                   |
| 创建 Skill 目录 | `Bash` 工具                                                   |
| 生成专属 Skill  | `Write` 工具                                                  |

**输出目录**：生成的读书笔记和 Skill 写入 `~/.opencode/skill/books/{slug}/`

## 核心流程：分析阅读 6 步法

### Step 0: 读取书籍内容

**根据文件类型选择读取方式**：

#### 方式一：`.txt` 文件
直接使用 `Read` 工具读取：
```
请读取书籍内容
```

#### 方式二：`.pdf` 文件
使用 `pdf_extractor.py` 转换为文本：`uv run python scripts/pdf_extractor.py --file "{书籍路径}" --output "./temp/{书名}.txt"`
然后用 `Read` 工具读取生成的 `.txt` 文件。

---

### Step 1: 书籍分类

**目标**：判断书籍类型，确定分析方法

**步骤**：
1. 读取 `prompts/step1_classify.md` 获取分类指导
2. 阅读书籍前言/目录/序言
3. 判断书籍类型：
   - **虚构小说类** → 阅读故事的方法
   - **理论性作品**（历史、科学、哲学、数学） → 关注观点与论证
   - **实用性作品**（方法论、自我提升） → 关注可执行的方法
   - **社会科学类** → 关注人与社会的关系

**输出**：记录书籍类型，后续分析以此为依据调整方法

---

### Step 2: 搭建骨架

**目标**：理解书籍整体结构

**步骤**：
1. 读取 `prompts/step2_skeleton.md` 获取骨架指导
2. 回答三个问题：
   - 这本书在谈什么？（用一句话概括）
   - 作者如何依次发展这个主题？
   - 作者想要解决什么问题？

3. 列出全书的重要部分：
   - 第一部分在讲什么
   - 第二部分在讲什么
   - ...

**输出**：
```
## 书籍骨架

### 一句话概括
[整本书在讲什么]

### 结构拆解
- 第一部分：[主题]
- 第二部分：[主题]
- ...

### 作者要解决的问题
- 问题1：
- 问题2：
```

---

### Step 3: 提取关键词 + 主旨句

**目标**：与作者达成共识，理解核心概念

**步骤**：
1. 读取 `prompts/step3_keywords.md` 获取提取指导
2. 找出书中反复出现的关键概念
3. 对每个重要单字，确认你对它的理解与作者一致
4. 找出作者最核心的主旨句（可用自己的话复述检验）

**输出**：
```
## 核心概念

| 概念  | 作者的定义 | 你的理解 |
| ----- | ---------- | -------- |
| 概念1 | ...        | ...      |
| 概念2 | ...        | ...      |

## 核心主旨
1. 主旨1：...
2. 主旨2：...
```

---

### Step 4: 评价作者

**目标**：形成自己的观点

**步骤**：
1. 读取 `prompts/step4_evaluate.md` 获取评价指导
2. 回答四个问题：
   - 这本书说的是真的吗？（同意/反对/部分同意）
   - 我同意/反对的理由是什么？
   - 这本书与我何干？
   - 读完后我有什么改变？

**输出**：
```
## 评价与思考

### 我的立场
- [ ] 同意 / [ ] 反对 / [ ] 部分同意

### 理由
- 同意的点：...
- 不同意的点：...

### 与我何干
- 收获：...
- 行动：...
```

---

### Step 5: 生成结构化读书笔记

**目标**：将分析结果整理为可复用的笔记

**步骤**：
1. 读取 `prompts/step5_note.md` 获取笔记模板
2. 整合 Step 1-4 的所有分析结果
3. 填充模板，生成完整读书笔记

**输出路径**：`~/.opencode/skill/books/{slug}/note.md`

---

### Step 6: 生成专属 AI Skill

**目标**：将书籍知识封装为可调用的 Skill

**步骤**：
1. 读取 `prompts/step6_builder.md` 获取 Skill 模板
2. 基于读书笔记，提取核心知识
3. 生成专属 Skill 文件

**输出路径**：`~/.opencode/skill/books/{slug}/SKILL.md`
