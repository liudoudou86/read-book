# 阅读的四个层次

> 基于《如何阅读一本书》整理

## 层次关系图

```
┌─────────────────────────────────────┐
│         主题阅读 (第4层)            │  ← 最主动、最复杂
│   针对一个主题，读多本书             │
├─────────────────────────────────────┤
│         分析阅读 (第3层)            │
│   全盘阅读，完整理解一本书           │
├─────────────────────────────────────┤
│         检视阅读 (第2层)            │
│   在有限时间内，抓出书的重点        │
├─────────────────────────────────────┤
│         基础阅读 (第1层)            │  ← 最基础
│   识字，能读懂句子                  │
└─────────────────────────────────────┘
```

**注意**：层次是递进的，高层次包含低层次的能力。

---

## 第一层：基础阅读

### 特征
- 认识字，能读懂句子
- 理解字面意思
- 小学阶段即可达到

### 在本 Skill 中的角色
- 作为默认已具备的能力
- 不需要特别处理

---

## 第二层：检视阅读

### 特征
- 在规定时间内完成
- 强调时间效率
- 目标是了解「这本书在讲什么」

### 适用场景
- 判断一本书是否值得读
- 快速了解一本书的框架
- 时间有限时的快速浏览

### 两种检视阅读

#### 1. 略读（有系统的粗读）
- 先看书名页，再看序言
- 研究目录页
- 看附有的索引
- 挑几个与主题相关的章节看
- 最后随便翻翻

#### 2. 粗浅的阅读
- 从头到尾先读完一遍
- 碰到不懂的地方不要停下来
- 重点放在看得懂的地方

### 在本 Skill 中的角色
- 在正式分析前，可先用检视阅读快速了解书籍
- 但本 Skill 聚焦分析阅读，检视阅读作为辅助

---

## 第三层：分析阅读

### 特征
- 在无限时间里，最好也最完整的阅读
- 特别在追寻理解
- 需要相当程度的技巧

### 适用场景
- 值得深读的好书
- 需要完全理解的书
- 需要从书中获取方法/知识

### 核心方法
详见 `analysis_rules.md`

### 在本 Skill 中的角色
- **核心层次**
- 本 Skill 主要处理的就是分析阅读

---

## 第四层：主题阅读

### 特征
- 读很多书，不是一本书
- 找出这些书之间相关之处
- 架构出一个可能在哪本书里都没提过的主题

### 最主动也最花力气

### 适用场景
- 深入研究某个主题
- 建立知识体系
- 解决特定问题

### 步骤
1. 找到相关章节
2. 带引作者与你达成共识（建立自己的术语）
3. 厘清需要解答的问题
4. 界定和确认议题
5. 分析讨论

### 在本 Skill 中的角色
- 作为进阶能力
- 可以基于多个已分析的书籍进行主题阅读
- 本 Skill 可扩展支持主题阅读

---

## 如何选择阅读层次？

| 需求 | 推荐层次 |
|------|---------|
| 快速判断书值不值得读 | 检视阅读 |
| 深入理解一本好书 | 分析阅读 |
| 研究某个主题 | 主题阅读 |
| 消遣娱乐 | 基础阅读 |
